# 🪨 Stone, Paper, Scissors Game 📄✂️

An interactive, responsive, and fun web-based **Stone, Paper, Scissors** game built using basic frontend technologies. Play against the computer and test your luck!



## 🚀 Features
* **Play Against AI:** The computer makes completely random and fair choices.
* **Real-time Score Tracking:** Keeps track of both the Player's and the Computer's scores instantly.
* **Dynamic Feedback Messages:** Displays clear, colorful messages showing who won each round (e.g., "You win! Rock beats Scissors").
* **Modern & Responsive UI:** Clean, minimalist design that adapts beautifully to both mobile phones and desktop screens.
